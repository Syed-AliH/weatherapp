# Weather Application

This weather application provides users with current weather information and a 5-day forecast for any specified location. The app utilizes the OpenWeather API to fetch weather data and the Gemini API for enhanced conversational interactions.

## Features
- **Current Weather**: Displays the current temperature, humidity, wind speed, and weather description.
- **5-Day Forecast**: Shows a detailed forecast for the upcoming days, including date, temperature, and weather conditions.
- **Chatbot Integration**: Users can interact with a chatbot to get additional information about the weather.
- **Responsive Design**: The app is designed to be responsive and works well on various devices.

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
- Gemini API (for chatbot functionality)

## Getting Started

### Prerequisites
- A web browser (e.g., Chrome, Firefox)
- Internet connection

### Installation
1. Clone the repository or download the source code.
2. Open the `index.html` file in your web browser.

### API Keys
- Obtain an API key from [OpenWeather](https://openweathermap.org/api) and replace it in the `tables.js` file.
- Obtain an API key from [Gemini API](https://developers.google.com/generative-language) and replace it in the `tables.js` file.

## Usage
- Enter a city name in the input field and click the search button to get the current weather and forecast.
- Use the chat interface to ask weather-related questions.
